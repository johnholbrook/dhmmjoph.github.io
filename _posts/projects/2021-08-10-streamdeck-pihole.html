streamdeck-pihole is a plugin for the Elgato Stream Deck that adds actions to monitor and control one or more instances of Pi-hole.

Features include:

  • Actions to toggle Pi-hole on/off or to temporarily disable Pi-hole for a short period of time.
  • Color of actions indicates whether the Pi-hole instance is enabled.
  • Optionally, display statistics from the Pi-hole instance (number of clients, percentage ads blocked today, etc.)

The plugin can be downloaded from the Stream Deck Store, and the source code is available on GitHub:
