FLLuid (FLL Useful Information Display) is an easy-to-use, customizable audience display package for FIRST LEGO League events run using FLLTournament.com.

FLLTournament provides an excellent for managing match scoring, judging, and volunteer roles. FLLuid extends this functionality by providing a variety of audience-facing displays for use around the tournament venue and in livestreams.

Features include:

  • Runs on Windows, macOS, and Linux
  • Suitable for use with or without streaming video
  • Match timer
  • Match scores display
  • Match schedule display
  • Dynamically-updating, smooth-scrolling tables
  • Show top scores from other qualifying events
  • Show sponsor logos on their own display, or in between match scores or schedule
  • Keep the audience up-to-date with custom messages
  • Customize the appearance and behavior of displays in various ways

FLLuid is configured and operated from the "controller window", which allows the user to configure the event, change displays, and control the match timer.

Audience-facing displays are served as web pages, which can be viewed in a browser on the same conputer as the controller, or any other machine on the same local network, making it easy to deploy displays throughout a tournament venue. Serving displays as web pages also makes it easy to incorporate them into OBS livestreams using the built-in browser source.
