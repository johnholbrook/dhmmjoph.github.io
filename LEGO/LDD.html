--- layout: LEGO title: LDD Sets ---

Because I am completely crazy, I've taken the time to convert various LEGO sets (just ones from LEGO Education so far) to LEGO Digital Designer, so you can build things from the set of your choice without physically having it. This is also convenient if you have the set and you wish to LDD a thing you have already built, since you don't have to go hunting around for the right part. Here are all the sets I've done so far:

EV3 Core Set (45544)

Download EV3_Core_45544.lxf

If you are using the rechargeable battery pack, note that it adds an additional beam-width to the EV3's height.
Since the LDD model does not include the battery pack (it uses the 6 AA batteries instead), you will need to keep this disparity in mind when using the EV3 in LDD.

EV3 Expansion Set (45560)

Download EV3_Expansion_45560.lxf

You can also download both the EV3 Core (45544) and Expansion (45560) sets as one LDD file, if you're building something which requires both sets.

NXT Base Set (9797)

Download NXT_Base_9797.lxf

If you are using the rechargeable battery pack, note that it adds an additional beam-width to the NXT's height.
Since the LDD model does not include the battery pack (it uses the 6 AA batteries instead), you will need to keep this disparity in mind when using the NXT in LDD.

NXT Resource Set (9695)

Download NXT_Resource_9695.lxf

You can also download both the NXT Base (9797) and Resource (9695) sets as one LDD file, if you're building a robot that requires both sets.
